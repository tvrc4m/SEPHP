
A.圈子
1.对于任何人都可创建圈子，圈子名称唯一，不可重复
2.相应的圈子有对应的类别，类别只有一级结构。
3.在创建圈子时，如发现圈子已创建，可以提醒用户去逛那个圈子，或者重新命名,必填:名称,类别,选填:描述信息，封面图片，标签,分享到公共平台(复选框)
4.创建圈子的形式有两种，一种可以是单独的页面，一种是弹出层。需要两个样式
5.圈子，可以有相应的操作，包括发表,加入,赞。
6.圈子里可以发表文字(文章),图片,视频,音乐(音频)，暂时只用文字(文章),图片 (导航栏上面图标形式)
7.分享文字(文章).必填：类别，标题,内容，选填：图片。(单独页面+弹出层)
8.分享图片，必填：类别，图片.选填：描述。(单独页面+弹出层)
9.对于分享,在列表页的操作有：赞,评论，在详细页，有赞,评论，转发,
10.分享详细页，有分享详情，分享的赞数量，评论数量，转发数量，点击数量，有评论功能，评论列表(分页),最新赞的人，最新转发的人，最近阅读的人。圈子相关分享推荐，猜你喜欢，热门推荐，左右靠边的翻页



主要页面有：有些情况要考虑瀑布流的形式，如果是瀑布流，就占用整个浏览器宽。
整个界面要有大气，简易之风，颜色上要稍微亮一点，但不能太艳。要考虑用户体验和视觉效果。
每个页面要尽量充实。
